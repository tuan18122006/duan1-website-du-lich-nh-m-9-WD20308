Các câu lệnh git


Lấy code từ máy chủ (nên dùng trước khi code)           =>          git pull origin <branch_name>

Kiểm tra các nhánh đã tồn tại                           =>          git branch

Tạo 1 nhánh mới                                         =>          git branch <branch_name>

Tạo và chuyển sang nhánh mới                            =>          git checkout -b <branch_name>

Chuyển sang nhánh khác                                  =>          git checkout <branch_name>


Cập nhật các file đã thay đổi                           =>          git add . (tất cả) / git add <file chỉ định> (1 file cụ thể)

Đẩy phần thay đổi lên Local repository                  =>          git commit -m "message"

Cập nhật lên server chính                               =>          git push origin <name_branch>


Gộp nhánh                                               =>          git checkout <nhánh muốn gộp vào>   -> git merge <branch_name>

Xem lịch sử commit                                      =>          git log

Xem thay đổi trước khi push                             =>          git diff
